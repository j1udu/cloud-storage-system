# 后端开发日志 - 2026/04/20

## 今日完成

### 1. MinIO 对象存储集成
- 实现 `storage/minio_client.go`：连接 MinIO，启动时自动创建 Bucket
- 实现 `storage/object_storage.go`：封装 PutObject、GetObject、RemoveObject、GetPresignedURL

### 2. 文件上传
- `model/file.go`：Matter 结构体（对应数据库 matter 表）+ 上传/列表响应 DTO
- `repository/file_repo.go`：Create、GetByID、ListByParent、CountByParent、UpdateName、UpdateStatus

- 重命名：更新数据库中的 name 字段
- 所有文件操作都验证 user_id，用户只能操作自己的文件

### 4. 工具函数
