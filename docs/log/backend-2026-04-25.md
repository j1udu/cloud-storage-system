## 遇到的问题

1. **parent_id=0 被拦截** — FolderCreateRequest 的 parent_id 标记了 `binding:"required"`，Gin validator 把数字零值视为空。根目录传 0 就校验失败。去掉 required 标签解决。

2. **Redis 缓存时间戳精度丢失** — 用 `time.RFC3339` 格式存取时间，毫秒被截断。改为自定义格式 `"2006-01-02T15:04:05.999Z07:00"` 保留毫秒。

3. **router.go 编辑失败** — 文件使用 CRLF 换行符和 tab 缩进，Edit 工具匹配不到原字符串。先用 `sed` 转换换行符再编辑。

4. **MySQL 时区问题** — 数据库连接 DSN 用的 `loc=Local`，服务器本地时间和北京时间有偏差。改为 `loc=UTC` 统一存 UTC，前端根据用户时区转换显示。
